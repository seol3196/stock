# 모의 주식 투자 프로젝트 계획서

## 1. 프로젝트 개요
- **목표**: 교실 내에서 교사가 학생들에게 경제 관념(주식, 저축)을 가르칠 수 있는 웹 기반 시뮬레이션 플랫폼 구축
- **타겟 장비**: Intel N100 탑재 Linux 미니 PC (서버용)
- **개발 환경**: Windows PC

## 2. 기술 스택 (Tech Stack)
- **프레임워크**: Next.js 14+ (App Router 방식)
  - React 기반의 풀스택 프레임워크로, 프론트엔드와 백엔드 API를 동시에 관리하기 용이함
- **언어**: TypeScript (안전한 데이터 처리를 위함) 또는 JavaScript
- **스타일링**: Vanilla CSS (CSS Modules)
  - 화려하고 동적인 디자인 구현을 위해 현대적인 CSS 활용 (Glassmorphism, 애니메이션 등)
- **데이터베이스**: SQLite
  - 파일 기반 데이터베이스로, 별도의 복잡한 설치 없이 미니 PC에서 구동하기 최적화됨 (이동 및 백업 간편)
- **ORM**: Prisma
  - 데이터베이스 관리를 쉽게 하기 위한 도구

## 3. 기능 상세 (Features)

### A. 시스템 관리자 (Admin) - 숨겨진 URL
1. **관리자 인증**: 별도 로그인 없이 특정 URL + 비밀키 등으로 접근
2. **교사 관리**:
   - 교사 계정 생성 (ID/PW 발급)
   - 모든 교사 페이지 접근 권한 (슈퍼유저)

### B. 교사 (Teacher)
1. **학생 관리**:
   - 학생 계정 일괄 생성/개별 생성
   - 학생 비밀번호 초기화
2. **주식 시장 관리 (Market Maker)**:
   - 종목 생성 (이름, 코드, 설명)
   - 가격 수정 (수동 시장 조작) - *방과후 활동*
   - 상장(IPO) 및 상장 폐지
3. **은행(중앙은행 역할)**:
   - 기준 금리 설정 (저축 이율)
   - 저축된 금액에 대해 이자 일괄 지급 기능
4. **학생 자산 개입**:
   - 특정 학생의 현금/주식 수량 강제 수정 (벌금, 보너스 등)

### C. 학생 (Student)
1. **대시보드**:
   - 총 자산 현황 (현금 + 주식 평가금 + 예금)
   - 수익률 그래프 및 표시
2. **주식 거래소**:
   - 매수/매도 주문 (시장가)
   - 실시간 가격 확인
3. **은행**:
   - 예금 입금/출금
   - 이자 수익 확인

## 4. 데이터베이스 구조 (Draft)

- **Teacher**: id, username, password
- **Student**: id, username, password, teacher_id, cash, savings_balance
- **Stock**: id, teacher_id, name, current_price, is_active (상장여부)
- **StockOwnership**: id, student_id, stock_id, quantity, average_buy_price
- **TransactionHistory**: id, student_id, stock_id, type(BUY/SELL), price, quantity, timestamp

## 5. 개발 및 배포 로드맵

### 1단계: 프로젝트 세팅 및 기초 공사
- Next.js 프로젝트 생성
- Prisma + SQLite 연동
- 기본 디자인 시스템 (CSS 변수, 컬러 팔레트) 구축

### 2단계: 핵심 기능 구현 (MVP)
- 로그인 시스템 (교사/학생)
- 교사 기능: 주식 생성, 학생 생성
- 학생 기능: 주식 매수/매도, 잔고 확인

### 3단계: 고급 기능 및 UI 고도화
- 저축/이자 시스템
- 대시보드 시각화 (차트, 수익률 등)
- 동적 애니메이션 및 스타일링 적용

### 4단계: 배포 (Deployment)
- Windows에서 빌드
- 리눅스 서버(N100)로 빌드 파일 전송 및 실행 스크립트 작성
- PM2 등을 이용한 무중단 서비스 설정
